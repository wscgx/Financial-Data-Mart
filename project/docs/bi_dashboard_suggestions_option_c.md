# 选项C：客户风险匹配度分析 - BI展示方案建议

## 1. 仪表盘布局建议

### 1.1 顶部KPI卡片区域
- **风险错配客户数**: 显示当前日期风险错配客户总数，环比变化
- **测评过期率**: 百分比显示，红色预警如果>10%
- **高风险预警数**: 显示alert_level='high'的预警数量
- **合规客户占比**: 百分比显示，绿色健康如果>90%

### 1.2 中部图表区域

#### 左侧：风险分布饼图
- **数据源**: dws_risk_compliance_summary
- **维度**: compliance_status (compliant/warning/violation)
- **指标**: 客户数量
- **用途**: 直观展示整体合规情况

#### 右侧：风险等级交叉矩阵热力图
- **数据源**: dwd_customer_risk_match
- **X轴**: 产品风险等级 (conservative → aggressive)
- **Y轴**: 客户风险等级 (conservative → aggressive)
- **颜色深浅**: 持仓客户数或持仓市值
- **用途**: 识别风险错配模式，对角线以下为错配区域

### 1.3 底部明细区域

#### 风险错配预警清单表格
- **数据源**: ads_risk_mismatch_alert
- **列**: 预警ID、客户姓名、客户风险等级、产品名称、产品风险等级、风险差距、持仓市值、预警级别、预警类型
- **筛选**: 支持按预警级别、预警类型、日期筛选
- **排序**: 默认按预警级别(高→低)、风险差距(大→小)排序
- **用途**: 合规人员逐条处理预警

## 2. 详细报表建议

### 2.1 客户风险画像报表
- **数据源**: dws_risk_compliance_summary
- **维度**: 客户风险等级、测评状态、合规状态
- **指标**: 客户数、总持仓市值、错配持仓数、错配比例
- **筛选**: 客户风险等级、合规状态、测评是否过期
- **用途**: 按风险等级分析客户群体特征

### 2.2 产品风险分布报表
- **数据源**: dwd_customer_risk_match
- **维度**: 产品风险等级、产品类型
- **指标**: 持有客户数、总持仓市值、错配客户数、错配市值占比
- **筛选**: 产品风险等级、产品类型
- **用途**: 分析各风险等级产品的销售合规情况

### 2.3 测评过期跟踪报表
- **数据源**: dws_risk_compliance_summary
- **维度**: 测评状态、测评有效期至(按月分组)
- **指标**: 客户数、持仓市值
- **筛选**: 测评状态、过期时间范围
- **用途**: 跟踪需要重新测评的客户

## 3. 预警推送建议

### 3.1 高风险预警实时推送
- **触发条件**: ads_risk_mismatch_alert中alert_level='high'
- **推送内容**: 客户姓名、产品名称、风险差距、持仓市值
- **推送对象**: 合规经理、客户经理

### 3.2 测评过期提醒
- **触发条件**: assessment_valid_until在未来30天内
- **推送内容**: 客户姓名、测评到期日期、当前风险等级
- **推送对象**: 客户经理

### 3.3 日报/周报
- **内容**: 风险错配客户数趋势、测评过期率趋势、预警处理情况
- **推送对象**: 合规部门负责人、业务主管

## 4. 可视化组件建议

| 图表类型 | 用途 | 数据源 | 维度 | 指标 |
|----------|------|--------|------|------|
| KPI卡片 | 核心指标展示 | dws_risk_compliance_summary | - | 错配客户数、过期率、合规占比 |
| 饼图 | 合规状态分布 | dws_risk_compliance_summary | compliance_status | 客户数 |
| 热力图 | 风险错配矩阵 | dwd_customer_risk_match | 客户风险×产品风险 | 持仓数/市值 |
| 折线图 | 趋势分析 | dws_risk_compliance_summary | stat_date | 错配客户数、过期率 |
| 表格 | 预警清单 | ads_risk_mismatch_alert | 明细 | 全部字段 |
| 柱状图 | 各等级客户分布 | dws_risk_compliance_summary | customer_risk_level | 客户数、AUM |
| 漏斗图 | 预警处理流程 | ads_risk_mismatch_alert | alert_level | 预警数、处理数 |

## 5. 交互功能建议

### 5.1 钻取功能
- 从合规状态饼图钻取到对应状态的客户清单
- 从热力图错配区域钻取到具体预警记录
- 从预警清单钻取到客户详细风险画像

### 5.2 筛选器
- 日期范围筛选
- 客户风险等级筛选
- 产品风险等级筛选
- 预警级别筛选
- 预警类型筛选
- 测评状态筛选

### 5.3 导出功能
- 预警清单导出Excel
- 报表导出PDF
- 自定义时间范围导出
