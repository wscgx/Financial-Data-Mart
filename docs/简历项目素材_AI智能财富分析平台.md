# 简历项目经历素材 - AI智能财富分析平台

## 项目基本信息

**项目名称**：AI智能财富分析平台（个人项目）

**项目描述**：基于大模型 Agent 技术，从0到1独立搭建的金融数据仓库项目，涵盖客户资产分析、产品销售分析、风险匹配度分析、交易行为分析及综合管理驾驶舱5大业务主题。

**技术栈**：MySQL + Python + SQL + AI Agent + ETL + 数据质量校验 + BI可视化

**项目时间**：2026.06 - 至今

---

## 项目职责（可直接复制到简历）

- **架构设计**：独立完成 ODS→DWD→DWS→ADS 四层数仓架构设计，制定表命名规范、DDL规范、ETL规范、Mapping文档规范、数据质量校验规范等开发标准
- **AI Agent 开发流程**：搭建 Master Agent + 专业子Agent（数据架构师、ETL开发工程师、指标设计师、数据质量工程师、BI分析师）的多智能体协作体系，实现需求分析→模型设计→ETL开发→质量校验→BI展示的自动化流水线
- **数据模型开发**：设计并实现6张ODS源表、多张DWD/DWS/ADS层目标表，覆盖客户资产、产品销售、风险合规、交易行为、管理驾驶舱等核心业务域
- **ETL脚本开发**：编写分层ETL SQL脚本，实现数据清洗、聚合、窗口函数计算、同比环比分析、多维度交叉分析、异常检测等业务逻辑
- **数据治理体系**：建立完整的数据质量校验机制（24+条校验规则），实现数据血缘追踪、字段映射文档（CSV+MD）、指标定义文档的自动化生成
- **全链路落地**：完成从 SQLite 到 MySQL 的数据库迁移，解决字符编码、数据类型兼容、SQL语法转换等技术问题，实现数据完整落地与注释验证
