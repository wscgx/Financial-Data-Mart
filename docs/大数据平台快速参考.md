# 大数据平台快速参考

## 常用命令

### 启动/停止服务
```bash
# 启动所有服务
.\启动大数据平台.bat

# 停止所有服务
.\停止大数据平台.bat

# 或使用 Docker Compose
cd project\docker
docker compose up -d
docker compose down
```

### 查看服务状态
```bash
cd project\docker
docker compose ps
```

### 查看日志
```bash
# 查看特定服务日志
docker compose logs -f spark-master
docker compose logs -f flink-jobmanager
docker compose logs -f kafka

# 查看所有服务日志
docker compose logs -f
```

## ETL 执行

### 手动执行 ETL
```bash
# 进入项目目录
cd project

# 执行全量 ETL
python scripts/run_all_etl.py

# 执行指定选项
python scripts/run_all_etl.py --option a b

# 仅执行指定层级
python scripts/run_all_etl.py --tier dwd dws

# 跳过 DDL
python scripts/run_all_etl.py --skip-ddl
```

### PySpark ETL (Delta Lake 版本)
```bash
# 在 Jupyter 中执行
# 或使用 spark-submit
spark-submit etl/spark/etl_ods_to_delta.py --date 2026-06-23
spark-submit etl/spark/etl_dwd_to_delta.py --date 2026-06-23
spark-submit etl/spark/etl_dws_to_delta.py --date 2026-06-23
spark-submit etl/spark/etl_ads_to_delta.py --date 2026-06-23
spark-submit etl/spark/etl_ads_sync_mysql.py --date 2026-06-23
```

## 常见问题

### 1. Spark 无法连接到 MinIO
检查 MinIO 是否正常运行:
```bash
docker compose ps minio
curl http://localhost:9000/minio/health/live
```

### 2. Kafka 消息发送失败
检查 Kafka 是否正常运行:
```bash
docker compose ps kafka
docker compose logs kafka
```

### 3. Airflow DAG 未加载
检查 DAG 文件是否挂载正确:
```bash
docker compose exec airflow-webserver ls /opt/airflow/dags
```

### 4. Jupyter 无法连接 Spark
检查 Spark Master 是否正常运行:
```bash
docker compose ps spark-master
curl http://localhost:8083
```

### 5. 内存不足
增加 Docker Desktop 内存限制 (推荐 8GB+):
- Docker Desktop → Settings → Resources → Memory: 8GB

## 数据流向图

```
┌─────────────────────────────────────────────────────────────┐
│                        数据源层                              │
│  MySQL (OLTP)  ←→  Kafka (实时流)  ←→  MinIO (数据湖)       │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        ETL 层                               │
│  ODS (原始数据) → DWD (明细层) → DWS (汇总层) → ADS (应用层) │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        计算引擎                              │
│  Spark (批处理)  |  Flink (流处理)  |  Spark Streaming      │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        存储层                               │
│  Delta Lake (MinIO)  |  MySQL (查询层)                      │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        应用层                               │
│  Streamlit (看板)  |  Grafana (监控)  |  Jupyter (分析)     │
└─────────────────────────────────────────────────────────────┘
```
